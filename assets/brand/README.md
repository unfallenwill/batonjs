# BatonJS Brand Mark

The mark combines three project ideas:

- **Baton**: the diagonal conductor baton represents orchestration.
- **Flow**: branching lines and nodes represent `parallel()`, `pipeline()`, and phases.
- **JS package**: the amber accent keeps the mark connected to the JavaScript ecosystem without using a generic JS badge.

## Assets

- `batonjs-logo.svg` — full lockup for README headers, docs, websites, and social previews.
- `batonjs-icon.svg` — square app/package icon for favicons, avatars, and compact UI.

## Palette

- Ink: `#0F172A`
- Blue: `#1D4ED8`
- Teal: `#0F766E`
- Amber: `#F59E0B`
- Surface: `#F8FAFC`
- Border: `#E2E8F0`

## Usage

Use the icon at small sizes and the full lockup where the wordmark needs to be readable. Keep at least one icon-node width of clear space around the mark.
